# Realistic Winch Simulator

**Status:** Early Development

Currently, this project features only a cable simulation. The goal is to develop a full winch simulator in Unity, aiming for the highest possible accuracy to real-world winch behavior.

## How to Run

1. Open the project in Unity.
2. Navigate to the `CableSimScene`.
3. Drag around the cable and experiment with the sockets and Cable Manager settings to explore the current simulation features.

---

Stay tuned for updates as the simulator expands beyond the cable system to a complete, realistic winch simulation.
